# Contribuição - Português

Qualquer contribuição é super bem-vinda!

Tentei fazer esse jogo para estudar mas tem alguns bugs que você pode ajudar a corrigir a aprender no processo durante a #hacktoberfest! Na dúvida, fale diretamente comigo pelo Twitter em [@juliabrazolim](https://twitter.com/JuliaBrazolim).
Para melhorar ou adicionar alguma coisa, é simples! Sinta-se à vontade :) Faça um fork, corrija, faça suas mudanças e depois só dar um pull request! Não sabe fazer nada disso? Aqui está um link para aprender: [Como criar seu primeiro pull request](https://youtu.be/Du04jBWrv4A).

### Teste

O projeto pode ser visto no codepen, mas se você cloná-lo em sua máquina, você pode testá-lo em seu navegador e vê-lo "funcionando"

# Contributing - English

Any contribution is very welcome!

I tried to make this game to study but it has some bugs you can help fix learning in the process during #hacktoberfest! When in doubt, contact me directly on Twitter at @juliabrazolim.
To improve or add something, it's simple! Feel free :) Fork, fix, make your changes and then just give a pull request! Can't you do any of this? Here is a link to learn: [Create Your First Pull Request](https://youtu.be/0BV1QCl8Az0)

### Test

The project can be seen in codepen, but if you clone it on your machine, you can test it in your browser and see it "working"
